To compile extrude.c, run the makefile

make

the executable will be named Extrude, to run the executable 

./Extrude

Base Features
Left Click points to draw polygon. Right click to extrude polygon for interative rotation,
by left clicking and dragging.

Additional Features
Press 'r' or 'R' to reset the extruded object to original rotation starting point
Press 'w' or 'W' to toggle between solid object, and wireframe object
Press 's' or 'S' to start over, from polygon drawing
	note: when restarting to polygon draw, the face will not draw as you draw, but 
		  right clicking will generate the correct extruded object
